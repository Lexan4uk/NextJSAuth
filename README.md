# NextJS + Prisma + SQLite + Redis Авторизация


## Общая информация

Проект авторизации на NextJS с сессиями через Redis и бд SQLite. Через логин/пароль и через OAuth2 (discord + github), с легкой типизированной возможностью расширения (см. /src/scripts/oauth).

Фронтенд-часть - формы авторизации и регистрации на react-hook-form с валидацией через неё же, личный кабинет с информацией о пользователе и возможностью переключения роли, прокси (в прошлых версиях Next миддлвер) перенаправляющий неавторизованных пользователей с /user на / для авторизации, страница admin, которая выдаёт 404 если у пользователя роль не admin.


Бэкенд-часть - бд на SQLite с prisma, менеджмент сессий в Redis (через сервис upstash), хранение сессий в cookies, хэширование паролей через crypto с хранящимся в бд salt, рабочая стандартизированная OAuth2 регистрация/авторизация без пароля с возможностью расширения (см. /src/scripts/oauth).
